Hello World!

From the top-level directory, run:

make
./bin/tex2web < example/hello.tex > hello.html

Then open up hello.html in a browser. You should see this file as clean and minimal HTML.

Alternatively, one can check the author's university web page for an example of this tool's output: http://www.csl.mtu.edu/~apklinkh/

1. Dependencies

The main dependencies of tex2web are the author's C utility library found here and the preprocessed version here. But don't worry about that, the make command does all the downloading for you.

2. Features

This tool does simple LaTeX formatting. Formats like bold, italic, and teletype are supported. as well as some custom macros.

  • inline code
  • Purpose-built formatting for -command-line-flags, filenames.h, symbols, LITERAL_VALUES, tool-names, and keywords.
    • Some of these may look the same, but it's nice to be explicit.

Quick Aside. Sometimes you want to make a quick section without a formal number or gigantic spacing. In this case, use \quicksec.
